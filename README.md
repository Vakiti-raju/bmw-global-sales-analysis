# BMW Global Sales Data Analysis

This project analyzes BMW global sales data to understand sales trends, regional performance, and electric vehicle adoption.

## Objectives
- Analyze BMW sales trends over time
- Identify top-performing regions and models
- Explore relationships between units sold and revenue

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Powe BI

## Key Insights
- BMW sales show a general upward trend.
- Certain regions contribute significantly to total revenue.
- Electric vehicle adoption is steadily increasing.

## Project Structure
- Data Cleaning
- Exploratory Data Analysis
- Visualizations
- KPI Summary
